# 婚礼邀请函网站 - 使用说明

## 项目简介

这是一套完整的婚礼电子邀请函系统，包含前台展示页面和后台管理系统。基于 Next.js 构建，支持全屏翻页浏览、背景音乐、视频播放、祝福留言、RSVP 回复等功能。

**前台地址：** `https://your-domain.com`  
**后台管理：** `https://your-domain.com/admin`

---

## 技术栈

| 技术 | 用途 |
|------|------|
| Next.js 16 | 全栈框架（SSR + API Routes） |
| React 19 | 前端 UI |
| Tailwind CSS 4 | 样式系统 |
| SQLite (better-sqlite3) | 数据库 |
| JWT + Cookie | 后台认证 |
| Sharp | 图片处理 |
| PM2 + Nginx | 生产环境部署 |

## 日常操作指南

### 修改婚礼信息

1. 进入后台 → 页面配置 → 婚礼信息
2. 修改新人姓名、婚礼日期、仪式地点等
3. 点击「保存」

婚礼日期修改后，前台首页倒计时会自动更新。

### 上传照片

1. 进入后台 → 甜蜜瞬间（或婚礼照片）
2. 点击「上传照片」选择图片文件
3. 拖拽调整显示顺序
4. 不需要的照片点击删除

### 更换背景音乐

1. 进入后台 → 背景音乐
2. 上传新的 MP3 文件
3. 点击「设为当前」切换播放曲目

### 查看 RSVP 回复

1. 进入后台 → RSVP
2. 可查看所有宾客的回复信息

### 配置收款码（发红包功能）

1. 进入后台 → 收款码配置
2. 上传微信收款码图片
3. 可选：配置弹窗提示文案（如"感谢您的心意 🧧"）
4. 上传后，前台祝福区会自动显示「发个红包」按钮

> 注意：长按识别二维码功能需要网站完成 ICP 备案后才能在微信中使用。

### 修改管理密码

1. 点击侧边栏底部「修改密码」
2. 输入当前密码和新密码
3. 确认修改

## 部署说明

### 服务器环境

- Linux 云服务器（阿里云 / 腾讯云等均可）
- Node.js 18+
- PM2 进程管理
- Nginx 反向代理 + HTTPS（Let's Encrypt）
- 已备案域名（微信长按识别二维码等功能需要 ICP 备案）

### 部署步骤

项目提供了 `deploy.sh` 部署脚本，使用前需修改脚本顶部的服务器地址和 SSH 密钥路径：

```bash
SERVER="root@your-server-ip"
KEY="~/.ssh/your-key.pem"
```

然后执行：

```bash
bash deploy.sh
```

脚本自动完成：
1. 本地构建 (`npm run build`)
2. 上传构建产物到服务器（rsync）
3. 确保服务器端原生模块正常（better-sqlite3, sharp）
4. 重启 PM2 服务

### 注意事项

- 数据库文件 `data.db` 在服务器上，部署不会覆盖
- `.env` 文件在服务器上，部署不会覆盖
- 上传的文件存储在服务器部署目录下的 `public/uploads/`

## 数据存储

所有数据存储在 SQLite 数据库中（`data.db`），包括：

- 网站配置（settings 表：婚礼信息、Logo、收款码等）
- 照片列表及排序
- 音乐列表
- RSVP 回复
- 祝福留言
- 地图点亮记录
- 访问统计数据

上传的文件存储在 `public/uploads/` 下：
- `photos/` - 甜蜜瞬间 & 婚礼照片
- `music/` - 背景音乐
- `video/` - 首页视频 & 婚礼邀约视频
- `milestones/` - 故事时间轴照片

## 特色功能说明

### 地图点亮

访客访问网站时，系统通过 IP 自动定位城市并在中国地图上点亮对应位置。地图采用四色光点系统：
- 粉红 - 婚礼地点（始终高亮）
- 绿色 - 出席宾客城市
- 金色 - 发送祝福者城市
- 蓝色 - 页面访客城市

### 背景音乐

采用悬浮黑胶唱片样式播放器，首次进入显示播放提示，支持静音/取消静音切换。

### iOS 微信兼容

系统已针对 iOS 微信浏览器做了专项优化：
- 检测 UA 自动禁用 scroll-snap（避免动画卡顿）
- 视频封面图解决首帧黑屏
- 交通地图禁用触摸交互（避免滚动冲突）

## 常见问题

**Q: 忘记管理密码怎么办？**  
A: 登录服务器，删除数据库中 `settings` 表里 `key = 'admin_password'` 的记录，系统会恢复默认密码 `admin`。

**Q: 微信里长按二维码没有"识别"选项？**  
A: 需要完成网站 ICP 备案。微信会对未备案域名禁用长按识别二维码功能。

**Q: 如何备份数据？**  
A: 登录服务器，复制部署目录下的 `data.db` 文件和 `public/uploads/` 目录即可完整备份。

**Q: 页面打开很慢？**  
A: 检查首页视频文件大小，建议使用 ffmpeg 压缩到 1.5Mbps 以下。图片上传后系统会自动压缩。

**Q: 如何修改页面的视觉风格？**  
A: 修改 `src/app/globals.css` 中的 CSS 变量（颜色、字体等），需要重新部署。

**Q: 如何适配自己的婚礼信息？**  
A: 部署后进入后台「婚礼信息」页面，修改新人姓名、日期、地点等即可，无需改代码。
